Zeo VR

What is Zeo VR?
Zeo VR is an open source project that lets you build WebVR worlds out of NPM plugins with your browser. The company behind it, Zeo VR Inc., supports the project with Zeo Hub subscriptions.
What do I need to run Zeo VR?
The only hard requirement is a web browser that supports WebGL -- there is full mouse + keyboard support. WebGL is supported every modern web browser.
Of course, to get the full experience, you'll want a desktop headset (HTC Vive or Oculus Rift/Touch) with a browser that supports WebVR. Currently that means the Chrome Experimental or Firefox Nightly, but the regular versions of Chrome and Firefox will soon ship with WebVR support out of the box.
Mobile WebVR (Chrome for Android) support is in the works. It works, but there's bugs, controller support, and optimization work to do.

Which NPM modules work with Zeo?
Any public NPM module with the zeo-mod keyword in its package.json keywords array will show up in Zeo VR. You can browse all such modules with npms.

Open Source

Is Zeo VR open source?
Yup, Zeo VR is released under the MIT license. You can find the source on Github.
Are Zeo NPM modules open source?
All of the Zeo modules we make at Zeo VR Inc. are open source (MIT licensed). While Zeo VR will install any module published to NPM with the zeo-mod keyword, we have no control over how those are licensed.

Zeo Hub

What is Zeo Hub?
Zeo Hub is the hosted version of Zeo VR. It's a distributed network of Zeo VR worlds managed by Zeo VR, Inc.
How much does Zeo Hub cost?
You can try Zeo Hub for free, but your Avatar, Adventures, Inventory, and Karma will not be saved unless you upgrade to an Admin account. Admin accounts are $20 per month.
What are Zeo Adventures?
Zeo Adventures are ideas for activities posted by your fellow VRnauts. By participating in Adventures you earn Karma. Zeo Adventures are time-limited to 15 minutes. At the end of an adventure each VRnaut rates the experience.
What are some examples of Zeo Adventures?
  • Scavenger hunt around the city
  • Build a log cabin
  • 3D paint charades
  • DJ booth dance party
  • Phychoanalyse my picture
What is Zeo Karma?
Zeo Karma are tokens of good will earned when you particupate in adventures with other VRnauts. You can use Karma to post your own Zeo Adventures.
Can I buy or sell Zeo Karma?
No, you cannot buy or sell Karma. This is prohibited in our Terms of Service. New accounts start with no Karma. The only way to get Karma is to participate in Zeo Adventures.
Do I need Zeo Hub to use Zeo VR?
Nope, you can host Zeo VR yourself: it's open source. If you do this you'll be running completely seperate from Zeo Hub.
What version of Zeo VR does Zeo Hub run?
Zeo Hub always runs the latest Zeo VR release automatically.
